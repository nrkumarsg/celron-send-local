4-Level Connectivity Matrix

1. Same WiFi (Local P2P)

Powered by WebRTC Direct Data Channels. Transfers happen strictly over your local router/hotspot at maximum speed. No cloud interaction ensures 100% data privacy for sensitive mission-critical documents.

2. Different Networks (Relay)

Utilizes STUN/TURN Bypass. This allows devices on separate networks, or behind restrictive corporate firewalls, to pair and share securely. Perfect for Ship-to-Shore communication where network conditions vary.

3. Global Device Connection

Orchestrated via Socket.io Real-time Signalling. Devices can be identified globally across 4G/5G, Starlink, or standard ISP networks, enabling seamless pairing with a simple 4-digit security code.

4. Offline Air-Gap Mode

Designed for the deepest engine rooms and 1,000 miles offshore. Functions via Private Portable Hotspots with ZERO internet required. 100% secure, 100% offline.
